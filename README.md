# SpendWise

SpendWise is a personal finance dashboard that helps users track income and expenses.

## Tech Stack
- FastAPI (Backend)
- PostgreSQL
- Vanilla JavaScript (Frontend)
- JWT Authentication
- Chart.js for analytics

## Features
- Track income and expenses
- Expense categories
- Financial analytics dashboard
- Secure authentication

## Project Structure
Backend/ – FastAPI server  
Frontend/ – UI using HTML, CSS, JS  

## Run Backend
pip install -r requirements.txt
uvicorn main:app --reload
